# Final Project

## Emotion Detector

Bu loyiha matndagi emotsiyalarni (quvonch, g'azab, qo'rquv, nafrat va xafalik) tahlil qilish uchun yaratilgan ilovadir.

### Loyiha maqsadi:
Ushbu loyihaning asosiy maqsadi - Watson NLP kutubxonasi yordamida foydalanuvchi kiritgan matnni tahlil qilish va undagi dominant emotsiyani aniqlash hamda natijalarni veb-interfeys orqali foydalanuvchiga taqdim etish.

### Asosiy vazifalar:
- `emotion_detector` funksiyasi yordamida matnni tahlil qilish.
- Flask serveri yordamida veb-ilovani joylashtirish (deploy).
- Unit-testlar orqali kodning to‘g‘ri ishlashini tekshirish.

### Ishga tushirish:
1. Loyiha papkasiga kiring.
2. Flask serverini ishga tushiring:
   ```bash
   python3 server.py
   ```
